Software Provider

Product

Software User

  • PK UserId + (FK) ProviderId

  • Email (UNIQUE)

License

  • PK LicenceId (AUTO)

  • FK ProductId + UserId

  • StartDate

  • EndDate (NULL)

  • HostId (NULL)
